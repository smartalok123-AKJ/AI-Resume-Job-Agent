# AI Resume Job Agent

An AI-powered Resume Analyzer and Job Matching Agent built using Python and Gemini API.

## Features

* Resume Skill Extraction
* ATS Score Analysis
* Job Description Matching
* Excel Report Generation
* AI Planning Agent
* Memory Agent
* Tool Calling Agent

## Technologies Used

* Python
* Gemini API
* Pandas
* NumPy
* Jupyter Notebook
* Excel
